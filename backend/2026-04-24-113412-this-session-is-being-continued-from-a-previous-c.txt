  ## Context                                                                    
                                                                                
  The user's previous formula `z_mm = level0 + slab вҲ’ depth/2` was meant to     
  place                                                                         
  the beam CENTROID at that elevation, relying on the Revit Add-in's            
  `Z_JUSTIFICATION = Center` to then land the beam top flush with the slab top  
  (at +slab_thickness). In practice, the Revit render puts the beam below Level 
  0 / column bases (see the user's screenshot), so `Z_JUSTIFICATION = Center`   
  is NOT landing the insertion Z on the centroid in this family вҖ” it's treating 
  the insertion Z as the beam TOP.                                              
                                                                                
  The user supplied a hand sketch clarifying the intended geometry:             
                                                                                
  - Columns sit ON Level 0 (column base at z = 0, top at Level 1).              
  - Slab sits ON Level 0 (slab bottom at z = 0, slab top at z =                 
  +slab_thickness).                                                             
  - Beam TOP is flush with the slab top surface.                                
                                                                                
  Given the observed "insertion Z = beam top" behavior of the current family,   
  the simplest formula that achieves this is the slab-top elevation directly,   
  with no depth offset.                                                         
                                                                                
  ## Recommended formula                                                        
                                                                                
  ```python                                                                     
  z_mm = level0_elev + slab_thickness                                           
  ```                                                                           
                                                                                
  With defaults (Level0 = 0, slab = 200, depth = 800):                          
                                                                                
  ```                                                                           
  z_mm        = 0 + 200   = +200 mm   вҶҗ beam top lands here                     
  beam bottom = +200 вҲ’ 800 = вҲ’600 mm  вҶҗ depth extends downward                  
  ```                                                                           
                                                                                
  Beam top coincides with slab top; beam body hangs into the foundation zone    
  below Level 0.

  glTF exporter at `backend/services/exporters/gltf_exporter.py:200` uses
